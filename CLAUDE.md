# Project: Git Practice

## เกี่ยวกับโปรเจกต์นี้
โปรเจกต์ทดลองสำหรับฝึกใช้ Git/GitHub Workflow

## กฎการเขียนโค้ด
- ใช้ภาษาไทยในการอธิบาย comment สำคัญ
- Commit message ใช้ภาษาอังกฤษ สั้น กระชับ
- ห้าม commit ตรงบน main โดยตรง ต้องสร้าง branch ก่อนเสมอ

## โครงสร้างโฟลเดอร์
- /docs — เอกสารประกอบ
- /rules — มาตรฐานการเขียนโค้ด
- /tasks — รายการงานที่ต้องทำ
